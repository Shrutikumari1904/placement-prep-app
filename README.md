# PlacementPrep

A web app to help students prepare for placement season — real interview questions from top companies, organized by difficulty level and topic, with a clean and easy-to-scan dashboard.

**Live Site:** https://placement-prep-app-black.vercel.app
**GitHub Repo:** https://github.com/Shrutikumari1904/placement-prep-app

---

## Features

- Company-wise breakdown of interview questions (e.g., Google, Amazon, Flipkart, and more)
- Color-coded difficulty tags — Easy / Medium / Hard — for quick scanning
- DSA topics to prepare, tagged per company
- "Projects They Value" section highlighting what companies actually look for
- Search and filter functionality across companies
- User authentication (Login/Signup) so users can access personalized prep
- Responsive design that works across devices

## Getting Started

### Prerequisites
- Node.js installed
- MongoDB connection string (if running the backend locally)

### Installation

```bash
# Clone the repo
git clone https://github.com/Shrutikumari1904/placement-prep-app.git
cd placement-prep-app

# Install frontend dependencies
cd client
npm install

# Run the development server
npm run dev
```

If there's a separate backend folder, also run:

```bash
cd server
npm install
npm start
```
